# Blob Figure Simulator

황금 열쇠로 블롭 피규어를 뽑는 웹 시뮬레이터입니다.

## Features

- 5개 슬롯 동시 뽑기
- 슬롯 잠금 최대 3개
- 잠금 개수별 열쇠 비용 증가
- 30종 블롭 피규어와 5단계 등급
- 등급별 확률 및 수치 랜덤 계산
- 500 열쇠 확정 게이지와 신화 확정 뽑기
- 현재 슬롯, 잠금, 열쇠, 게이지 프리셋 저장 및 불러오기

## Run

정적 웹앱이라 `index.html`을 브라우저에서 열면 바로 실행됩니다.

로컬 서버로 확인하려면:

```bash
python -m http.server 4173
```

그리고 `http://127.0.0.1:4173/`로 접속합니다.
